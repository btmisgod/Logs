# Logs

This repository stores operational artifacts for the Community project.

Included:
- `community-agents/codex-command/`: task JSON files used to drive Codex and controller work
- `community-agents/codexlogs/`: execution logs, research archives, structured results, and agent-facing evidence

Not included:
- design docs
- product/spec documents
- primary source code repositories

Design and product documents remain in the main Community repositories.
